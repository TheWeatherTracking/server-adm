Информация

Общее

С помощью этого интерфейса можно запустить kotlin-приложение, которое запустит N корутинов, в каждом из которых будет публиковать сообщения в соответствующие топики. Название топика = /devices/evt/{dev_name}. Пишет случайные данные. Каждый корутин каждую итерацию спит какое-то случайное время.

Важно

Взаимодействие с kotlin-приложением происходит через файл. "Остановить устройства" вырубает процесс приложения с корутинами. "Очистить лог" очищает лог-файл, через который ведется взаимодействие. После нажатия на "Запустить устройства" лучше не перезагружать страницу, а использовать только кнопку обновления.
